

Name of Project: 15-112 SUDOKU!! By Chris Bernitsas

How to run: The user should run the main.py file in the editor to launch the project

No Additional Libraries are required except for cmu_graphics

Shortcut Commands:

	A full list of shortcut commands can be found in the Help Screen which is easily accessible once launching the screen.
	Here are a few important ones:
		'p' to start the game
		'e' for an easy board, 'm' for a medium board, 'h' for a hard board, 'x' for an 'expert' board, and 'v' for an evil board
		
		The app can be fully played both with only the keyboard and only the mouse. Further shortcuts can be found in the Help Screen by 		pressing z once the playing screen is launched.
